2. 자바 프로그래밍을 위한 변수와 데이터형
================================

# 2.1 변수 제대로 알고 넘어가자
- Statically typed language
    - 컴파일할 때, 선언한 데이터 타입을 기준으로 컴파일러가 변수 선언에 에러에 대해서 검사
- Dynamically typed language
    - 프로그램을 실행할 때
- 변수명 규칙
    - 특수문자 : '_' 와 '$' 만 가능
